# Contributing To The CivitasX Parent Record

CivitasX is a parent framework and public continuity record for a long-term
civilizational direction.

This repository is not the product backlog for OrgAnchor, EchoEnginer,
ProvisionLoop, or WillFlow. Detailed subproject work belongs in the
corresponding subproject.

## Useful Parent-Level Contributions

Useful contributions include:

- correcting a material wording error or broken public link;
- improving a translation while preserving meaning;
- preserving, mirroring, archiving, or citing the public record;
- making the parent direction easier to understand or retell accurately;
- documenting a serious historical, scientific, ethical, or institutional comparison;
- improving the continuation path for later contributors;
- identifying a conflict between current public records;
- proposing a better parent-level mechanism while clearly distinguishing stable goals from replaceable design.

## Contributions That Belong Elsewhere

Use the relevant subproject for:

- OrgAnchor protocol, CLI, schema, evidence, or adopter issues;
- EchoEnginer product, interface, publishing, or workflow issues;
- ProvisionLoop engineering, robotics, energy, maintenance, or site design;
- WillFlow simulation, mechanism parameters, threat models, or experiment code.

Do not use this repository for token sales, investment promotion, political
campaigning, personality worship, mass marketing, or claims that unfinished
mechanisms are already proven.

## How To Contribute

Use a GitHub issue or pull request when available.

A useful proposal should state:

```text
What is wrong or missing?
Which parent-level purpose does it affect?
What evidence or reasoning supports the change?
Does it change a stable value, a confirmed structure, a candidate mechanism,
or only the wording?
```

Do not include passwords, recovery codes, private keys, account credentials,
private contact data, or confidential organizational information.

## Continuity And Forks

Anyone may create an adaptation, translation, independent continuation, or
fork under the public license.

State the relationship honestly:

```text
official signed continuation;
independent continuation;
historical fork;
translation;
adaptation;
or inspired work.
```

Do not claim official signed continuity without a valid OrgAnchor migration or
matching authority chain.
